# israel-polls-2026 plugin

See the [repository README](../../README.md) for full documentation.

This plugin bundles one skill, `israel-polls-dashboard`, which installs the live Hebrew polls dashboard as a Cowork artifact. The dashboard HTML lives at `skills/israel-polls-dashboard/assets/dashboard.html`.

After installing the plugin, ask Claude: "Install the Israel polls dashboard" / "התקן את לוח הסקרים".
